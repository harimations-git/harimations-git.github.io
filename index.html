Projects

MUSH | Handcrafted 2D Unity Adventure

Developed and designed MUSH, an in-progress 2D Unity game in C#, combining responsive player controls, state-based gameplay systems, URP 2D visuals, and handcrafted Tilemap levels.

HEdClass | Educational Management System

HEdClass is a full-stack educational management system built to make complex university workflows feel more streamlined and intuitive. Using JavaScript, Node.js, Express, MySQL, and EJS, it manages academic structures, user roles, and student classifications through secure backend logic, relational database design, role-based access control, and dynamic SQL-driven reporting.

Simple 3D Render Engine | Real-Time Java Graphics Demo

Simple 3D Render Engine is a lightweight 3D graphics project built in pure Java with Swing, designed to turn mathematical rendering concepts into something visual and interactive. It uses matrix transformations, 3D projection, Z-buffering, and surface normal lighting to render real-time 3D objects with live rotation and visual feedback.
